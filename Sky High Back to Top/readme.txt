== Description ==
Sky High Back to Top adds a beautiful, lightweight, and customizable “Scroll to Top” button to your WordPress site. Smooth animation, modern design, and zero bloat.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/sky-high-back-to-top` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Enjoy!
